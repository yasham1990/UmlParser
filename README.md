# UmlParser

General Overview
=======================
A Parser which converts Java Source Code into a UML class Diagram. Diagram generated is converted into a PNG format image
which can be used in blogs, forums, bug-trackers and email.

Technologies Covered
=======================
Java, yUml, ANTLR
